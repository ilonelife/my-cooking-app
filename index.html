🍳 Cooki 앱 개인정보처리방침

🔒 개인정보 보호 약속
Cooki 앱은 사용자의 개인정보를 소중히 여기며, 최소한의 정보만 수집하여 안전하게 관리합니다.

1. 수집하는 개인정보

📧 필수 정보

  • 이메일 주소: 계정 생성 및 로그인용

🚫 수집하지 않는 정보

  • 실명, 주민등록번호
  • 전화번호, 주소
  • 결제 정보 (무료 앱)
  • 위치 정보
  • 연락처, 사진, 파일 접근

2. 개인정보 이용 목적

  • 🔐 사용자 계정 관리 및 인증
  • 📱 앱 서비스 제공
  • 🛠️ 기술적 지원 및 문의 응답

3. 개인정보 보관 및 이용 기간

  • 계정 정보: 회원 탈퇴 시까지
  • 요리 기록: 사용자가 삭제하기 전까지
  • 자동 삭제: 3년간 미접속 시 자동 삭제

4. 개인정보 제3자 제공

🚫 제3자 제공 없음
Cooki는 사용자의 개인정보를 외부 업체나 제3자에게 제공하지 않습니다.

5. 개인정보 처리 위탁

📊 위탁 업체

  • Google Firebase
    • 위탁 업무: 사용자 인증, 데이터 저장
    • 보관 기간: 서비스 이용 기간
    • 위탁 근거: 안전한 클라우드 서비스 제공

6. 사용자 권리

🔧 사용자가 할 수 있는 것

  • 열람 권리: 내 정보 확인
  • 수정 권리: 잘못된 정보 수정
  • 삭제 권리: 계정 및 데이터 삭제
  • 처리 정지 권리: 개인정보 처리 중단 요청

📱 앱 내에서 할 수 있는 것

  • 프로필 화면에서 계정 정보 확인
  • 설정에서 데이터 삭제
  • 앱 삭제로 모든 로컬 데이터 제거

7. 개인정보 보호 기술적 조치

  • 🔐 암호화 전송: HTTPS/TLS 통신
  • 🛡️ Firebase 보안: Google 클라우드 보안 시스템
  • 🔒 접근 제한: 최소 권한 원칙
  • 📊 로그 관리: 개인정보 접근 기록 관리

8. 쿠키 및 추적 기술

🚫 추적 기술 미사용
Cooki 앱은 쿠키, 웹 비콘, 추적 픽셀 등을 사용하지 않습니다.

9. 아동의 개인정보 보호

  • 👶 13세 미만: 부모 동의 필요
  • 🔒 아동 정보: 추가 보호 조치 적용
  • 📧 부모 권리: 자녀 정보 열람/삭제 요청 가능

10. 개인정보처리방침 변경

  • 📅 변경 시: 앱 내 공지 또는 이메일 알림
  • 📖 확인 방법: 앱 설정 > 개인정보처리방침
  • 🔄 동의 철회: 언제든지 계정 삭제 가능
